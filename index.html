Typography

Breeze CSS implements a rem based font size system, using a root size of 10px. With the html font size set at 10px, all font-sizes can be globally changed by adjusting the html font size. Out of the box, Breeze CSS uses a base 10 system, so 5.0rem is equal to 50px.

Breeze CSS uses Raleway, which is served by Google's servers. Included are font weights 400, 500 and 600, as well as strong, emphasis and underline

  • Button elements

Forms

Forms are given basic styling, as well as inputs, selects, textarea and more.

For each input you have, wrap it in a label. Inside, include a span with a class of .label-body, which will contain your label. Next, include your input.

Tables

Tables are used for displaying data, and they shouldn't ever be confusing. That's why the default table in Breeze CSS is simply designed. It's full reponsive, and adapts to all browser and mobile phone sizes.

There are two flavors of tables: standard and dark.
