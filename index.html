Typography

Breeze CSS implements a rem based font size system, using a root size of 10px. With the html font size set at 10px, all font-sizes can be globally changed by adjusting the html font size. Out of the box, Breeze CSS uses a base 10 system, so 5.0rem is equal to 50px.

Breeze CSS uses Raleway, which is served by Google's servers. Included are font weights 400, 500 and 600, as well as strong, emphasis and underline

Forms

Forms are given basic styling, as well as inputs, selects, textarea and more.

For each input you have, wrap it in a label. Inside, include a span with a class of .label-body, which will contain your label. Next, include your input.

Lists

Lists are given basic styles to make them easy to read and understand. Both unordered lists and ordered lists support nested list. each nested list is given a different list style to distinguish them from their parents.

    List titles are used inside each list

  • Unordered lists are styled to be clean and easy to read.
  • They also support nested list.
    • Nested lists can be both ordered and unordered.
    • Third-tier lists are also supported in Breeze CSS
      • And even third-tier lists
      • That's totally cool!
  • I'm not as cool as the third-tier list.

    Ordered lists are awesome, too!

  1. Ordered lists have the same styling as unordered, except for the bullets on the front.
  2. Ordered lists also support nested lists
    • Nested lists can be both ordered and unordered.
    • Third tier lists are also supported.

Tables
